Software Engineering/Design

This artifact represents several software design documents that are blueprints for a Student Information System at a university. This was first developed in the Fall of 2019.

I opted to include this in my portfolio because it my promotes my ability to design a system based on business cases while industry standard communication so other readers can understand the problem that is is solving. Specific components that showcase these skills is the use of the Unified Modelling Language (UML). This is a design language that is specifically used in the field of software engineering. This increases the ability to communicate youre ideas in a language that other software engineers will understand. Another skill showcased is the ability to adapt a business case to a specific design diagram. Having knowldege about the design templates availible to me and being able to adopt them to a problem I am solving.

As I was building these design documents, I learned that design is heurustic. Meaning there is no right answer when it comes to designing systems. But there are designs that are more efficient than others.

Algorithms & Datastructure

This artifact represents my ability to enhance an already exiting algorithm to make it more pleasant for the user and the developer. This project was originally developed in Summer 2019.

The reason why this was included in the porfolio is because it represents my ability to use ehance code using basic algorithms. Using a java library to manipulate files and implement an algorithm to display each file as a menu that users can select from. Doing this took some research about know what is possible in the language and how it can be used to make the application better.

The process for making enhancements was by using visual inspection of the code to see where more value can be added. As I was readling the code line by line, I was able to find an area where an enhance can be made. After all the improvements, this artifact now has a menu that prints the playlist on the juke box, addititonal playlist added, test code to validate the jukebox, and additional comments.

Database

This artifact represents the ability to use a back-end database framework to support simple web application. This project was created during the Summer of 2020.

I selected this project because it was a new challenged for me to overcome and learn something new. The compoments that showcase my database ability to was implement the Django framework and use several basic commands to interact with the database.

The process was diffucult as I was not familiar with the Django framework so it was framework that I was stumbling through. Even though I found this challenging, it was more productive to start a new project than to go back and take time to update a project that was previously created. I believe overall that this artifact was an improvement in my ability to work with databases. Being able to employ a popular piece of technology is an important skill for software engineers. This is because it increases productivity and keep the focus on solving the problem.
